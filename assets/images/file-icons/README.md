# Iconos de Archivos

Esta carpeta contiene los iconos de los diferentes tipos de archivos que se muestran en la aplicación.

## Iconos requeridos:

1. **word.png** - Icono de Microsoft Word (azul #2B579A)
2. **excel.png** - Icono de Microsoft Excel (verde #217346)
3. **powerpoint.png** - Icono de Microsoft PowerPoint (naranja #D04423)
4. **pdf.png** - Icono de Adobe PDF (rojo #DC143C)

## Recomendaciones:

- Formato: PNG con transparencia
- Tamaño recomendado: 64x64 píxeles o 128x128 píxeles
- Los iconos deben tener sus colores originales de las aplicaciones
- Puedes descargarlos desde:
  - [Flaticon](https://www.flaticon.com/)
  - [Icons8](https://icons8.com/)
  - [IconFinder](https://www.iconfinder.com/)
  - O cualquier otra fuente de iconos de archivos

## Nota:

Una vez que agregues los archivos PNG en esta carpeta, la aplicación los cargará automáticamente.
